# Model-Fitness-Churn-Analysis
Customer churn prediction

## Project Description
This project analyzes customer data from a fitness club to identify factors related 
to customer churn and provide data-driven recommendations to improve retention.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Scikit-learn

## Key Analysis
- Exploratory data analysis of customer behavior
- Comparison between churned and retained users
- Feature importance analysis
- Customer segmentation

##  Conclusion
The analysis identifies key behavioral and demographic factors associated with churn. 
Based on these insights, actionable recommendations are proposed to improve customer 
retention and engagement.
